## Feature Overview

- Flutter (Dart)
  - App/Web UI
  - Camera/Video
  - Darkmode
  - Navigation & GoRouter

- Firebase
  - Authentication (signup, login, signout)
  - Database, Storage : save and upload contents
  - Filtering (serch users)
